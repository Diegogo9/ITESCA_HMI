# ITESCA_HMI
Proyecto de residencias profesionales  
Sistema de Interfaz Humano Maquina usando python Tkinter  



Probado en micropython con:  
- Raspberry Pi Pico (RP2040)
- ESP-WROOM-32 
- ESP32S2 

Implementando :
- 4 Entradas Digitales
- 2 Entradas Analogas
- 4 Salidas Digitales
